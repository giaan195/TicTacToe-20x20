### ĐỒ ÁN GAME CARO ONLINE WITH PYTHON3 ###

Python Version: 3.10.9
Thư viện sử dụng:
    + socket: Tạo socket server, trao đổi giữ liệu realtime giữa các client và server
    + tkinter: Tạo giao diện, Messagebox để hiện thông báo
    + motor: Sủ dụng mongodb
    + threading: Chia luồng xử lý
    + json: Chuyển đổi dữ liệu trước khi gởi và sau khi nhận (Socket)
    + PIL(PIP) : Xử lý hình ảnh
